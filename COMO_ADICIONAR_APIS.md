# Como adicionar as chaves de API

Adicione estas linhas ao arquivo `.env` na raiz do projeto `codex-IA`:

```bash
# Existentes
GEMINI_API_KEY=sua_chave_aqui

# NOVAS - Adicione conforme conseguir
MISTRAL_API_KEY=sua_chave_mistral
ANTHROPIC_API_KEY=sua_chave_claude  
COHERE_API_KEY=sua_chave_cohere
HUGGINGFACE_API_KEY=sua_chave_huggingface
PERPLEXITY_API_KEY=sua_chave_perplexity

# Opcionais (já suportadas)
OPENAI_API_KEY=sua_chave_openai
GROQ_API_KEY=sua_chave_groq
XAI_API_KEY=sua_chave_grok
DEEPSEEK_API_KEY=sua_chave_deepseek
```
