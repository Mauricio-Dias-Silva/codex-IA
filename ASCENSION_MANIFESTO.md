# CODEX-IA: THE ASCENSION MANIFESTO
> "The code that writes itself."

## 1. Project Overview
Codex-IA is an advanced, agentic AI coding environment designed to evolve autonomously. It combines a modern Electron/React desktop interface with a powerful Python-based AI brain (`CodexAgent`), capable of executing complex missions, self-improving via "Night Shift", and integrating with the PythonJet ecosystem.

### Current Version: 2.0.0 (Ascension Phase)

---

## 2. Architecture

### The Trinity
The system is composed of three main pillars:

1.  **The Interface (Frontend)**
    *   **Tech:** Electron, React, Vite, TailwindCSS, Monaco Editor.
    *   **Role:** Provides a VS Code-like experience, manages project state, and visualizes AI operations.
    *   **Communication:** Communicates with the backend via IPC (Inter-Process Communication) over stdin/stdout.

2.  **The Brain (Backend)**
    *   **Tech:** Python 3.11+, LangChain, Google Gemini Pro.
    *   **Role:** Hosts the `CodexAgent`, `SquadLeader` (Level 8), and `EvolutionAgent` (Level 5/9).
    *   **Key Components:**
        *   `app.py`: The bridge between Electron and Python.
        *   `codex_ia.core.agent`: The central intelligence.
        *   `codex_ia.core.squad`: Orchestrates multi-agent missions (Coder, Tester, Researcher).

3.  **The Cloud (PythonJet)**
    *   **Role:** Deployment target and verification platform.
    *   **Integration:** Codex-IA can deploy projects directly to PythonJet (Cloud Run).

## 5. Deployment & Setup

### Building the Executable
1.  Navigate to `codex-desktop-v2`.
2.  Run `npm install`.
3.  Run `npm run build` (Build React).
4.  Run `npm run dist` (Package Electron).
5.  Find executable in `dist/win-unpacked/Codex-IA.exe`.

### Python Requirements
The app requires a local Python environment with:
*   `langchain`, `google-genai`, `chromadb`, etc.
*   Dependencies listed in `codex_ia/requirements.txt`.
